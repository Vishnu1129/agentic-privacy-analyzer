# 🔐 Agentic Privacy Analyzer

An Agentic AI-based web application that analyzes a user's digital exposure and provides actionable privacy recommendations.

## 🚀 Features
- Multi-agent architecture
- Data Finder Agent (checks exposure)
- Risk Analyzer Agent (assigns risk level)
- Privacy Advisor Agent (gives recommendations)
- Deterministic results (same input → same output)
- Risk visualization
- Downloadable report
- Streamlit UI

## 🧠 Architecture

User Input → Data Finder → Risk Analyzer → Privacy Advisor → Output

## 🛠 Tech Stack
- Python
- Streamlit
- Modular Agent Design

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py

🎯 Use Case

This system helps users understand their digital footprint exposure and take steps to improve privacy and security.
